环境准备

在开始之前，请确保已安装以下工具：

  1. Node.js：Hexo 基于 Node.js，因此需要先安装它。

    • Node.js 官方下载

  2. Git：用于版本控制和将博客部署到 GitHub。

    • Git 官方下载

  3. Visual Studio Code：强大的代码编辑器，用于撰写和管理博客文章。

    • VS Code 官方下载

vscode笔记辅助插件

  • Markdown All in One ：提供 Markdown 语法高亮、预览等功能。
